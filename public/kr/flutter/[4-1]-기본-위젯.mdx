# Container

디자인을 위한 네모 박스라 생각하면 됩니다. 웹개발을 해봤다면 div 태그와 비슷하다 생각하면 됩니다. 하지만 주의할점은 flutter 의 디자인 시스템은 html, css 와 는 다른점이 많다는 점만 주의하면 되겠습니다.

만약 이러한 개발을 처음이라면 한번 자주 방문하는 웹사이트나 앱을 켜보고 자세히 보면 디자인이란게 사실 네모박스, 그리고 네모박스 내부의 이미지와 텍스트, 이러한 네모박스 안의 이미지와 텍스트를 정렬, 사실 이것 뿐인걸 알 수 있음, 이밖에도 그림자 같은 부가적인 것들이 있지만 진짜 말그대로 부가적인거고 대부분 저 3 개 면 기본적인 디자인은 전부 만들 수 있다 생각해도 됩니다.

너무 다 외울필요는 없고 나중에 데이터를 다룰때 집중하는걸 추천드립니다. 이건 그냥 한번 읽고 이런게 있구나 정도면 충분할듯 싶습니다.

Container 에는 여러 옵션들이 있지만 대부분 디자인적인 요소를 위한 것이라고 생각하면 됩니다.

```bash
(new) Container Container({
  Key? key,
  AlignmentGeometry? alignment,
  EdgeInsetsGeometry? padding,
  Color? color,
  Decoration? decoration,
  Decoration? foregroundDecoration,
  double? width,
  double? height,
  BoxConstraints? constraints,
  EdgeInsetsGeometry? margin,
  Matrix4? transform,
  AlignmentGeometry? transformAlignment,
  Widget? child,
  Clip clipBehavior = Clip.none,
})
```

여기서 자주 쓰게 될것들은 color, width, height, padding, margin, decoration 정도 일겁니다.

## width, height
말 그대로 각각 가로 세로 길이를 설정하는 옵션 입니다. 그냥 숫자로 넣어도 잘 작동하지만, 대부분의 경우, 이렇게 고정된 길이를 주는것보단, 비율로 길이를 주고 싶을 때가 많죠, 이럴땐 추가적인 Widget 이 필요합니다.

그리고 Container 는 기본적으로 부모 위젯의 길이에 맞춰 길이가 설정됩니다. 즉 부모 위젯이 가로 100 이라면 자식 위젯도 가로 100 이고, 부모 위젯이 가로 50 이라면 자식 위젯도 가로 50 이라는 것입니다. 그래서 Scaffold 의 body 에 넣어주면 부모 위젯이 Scaffold 가 되기 때문에 Scaffold 의 길이에 맞춰서 길이가 설정됩니다.


```dart:Container-width-height
```



width 와 height 는 각각 가로 세로 길이를 설정하는 옵션임, 그냥 숫자로 넣어도 잘 작동하지만, 대부분의 경우, 이렇게 고정된 길이를 주는것보단, 비율로 길이를 주고 싶을 때가 많음

예를들면 가로 100% 라든가.. 이럴땐 double.infinity 를 넣어주면 됨, 이건 무한대라는 뜻임, 즉 가로 길이를 무한대로 설정하겠다는 뜻임, 이걸 넣어주면 부모 위젯의 길이에 맞춰서 가로 길이가 설정됨 여기서 포인트는 비율로 길이를 준다면 그 길이는 부모 위젯의 길이에 맞춰서 설정된다는 것임, 즉 부모 위젯이 가로 100 이라면 자식 위젯도 가로 100 이고, 부모 위젯이 가로 50 이라면 자식 위젯도 가로 50 이라는 것임, 즉 부모 위젯의 길이에 맞춰서 자식 위젯의 길이가 설정됨

하지만 예를들어 부모 위젯의  `2/3` 만 사용하고 싶다면 예기가 좀 복잡해짐, LayoutBuilder 를 사용해야함, 이건 부모 위젯의 길이를 가져와서 그 길이에 맞춰서 자식 위젯의 길이를 설정 가능하게 해주는 위젯임 사용 방법은 아래와 같음


```dart
LayoutBuilder(
  builder: (context, constraints) {
    return Container(
      width: constraints.maxWidth * (2 / 3),
      height: 100,
      color: Colors.orange,
      child: Text('부모 너비의 2/3'),
    );
  },
)
```

이때 contraints 가 중요한 개념인데 지금은 단순 네모 박스인 Container 만 다루겠음, 바로 다음 Layout 을 다룰때 자세히 다루겠

그리고 decoration 또한 자주 사용하게 될 옵션임 이 옵션을 쓰면 Container 에 그라데이션, 그림자, 테두리 등을 줄 수 있음

```dart:Container-decoration
```

## Card

카드는 Material Design 에서 제공하는 카드 위젯임, 이 위젯은 기본적으로 그림자와 둥근 모서리를 가지고 있음, 이 위젯은 주로 리스트 아이템이나 카드 형태의 UI 를 만들때 사용됨

사실 그냥 모서리가 둥글고 그림자가 기본적으로 적용된 사용자 지정 옵션이 적은 Container 라고 생각하면 됨, 그래서 Card 를 사용하면 Container 를 사용할 때보다 더 간편하게 고급스러운 UI 를 만들 수 있음, 그뿐임임

```dart:Card
```
