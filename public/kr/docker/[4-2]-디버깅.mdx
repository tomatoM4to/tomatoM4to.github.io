# Container 디버깅

여기서 말하는건 코드의 디버깅이 아닙니다.

만약 백엔드 서버를 배포 했는데 문제가 생겼다면 일단, 코드에 문제가 없는지 판단하고 그 다음 서버에 문제가 있는지 확인해야 합니다.

Container 를 이용한 배포또한 마찬가지 입니다. Container 자체가 잘못된 상황이 분명 발생 할 수 있습니다. 그래서 Container 의 내부 상태를 이해하고 조작하는 방법을 알아야 합니다.

혹은 Nginx 나 PostgreSQL 같은 Container 를 배포했을때, 내부 파일을 수정해야 할 때도 있습니다.
