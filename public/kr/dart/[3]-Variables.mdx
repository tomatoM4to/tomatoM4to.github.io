# var Keyword

**Dart** 는 기본적으로 `var` 키워드를 사용하여 변수를 선언합니다. **Dart** 는 정적 타입 언어지만 `var` 키워드를 사용하면 타입을 명시하지 않고 변수를 선언할 수 있습니다.

하지만 그렇다고 **Type** 이 없다는 뜻이 아님니다. <R>Dart 는 타입 추론을 통해 변수의 타입을 자동으로 결정합니다.</R> 예를 들어, 아래와 같이 변수를 선언할 수 있습니다.

```java:main.dart
void main() {
  var name = 'John Doe'; // String
  var age = 30; // int
  var height = 1.75; // double
  var isStudent = false; // bool

  print('Name: $name');
  print('Age: $age');
  print('Height: $height');
  print('Is Student: $isStudent');
}
```

vscode 에서 name 변수에 마우스를 올리면 `String` 이라는 타입을 확인할 수 있습니다.

# 명시적 타입 선언

**C++** 나 **Java** 처럼 명시적으로 타입을 선언할 수 있습니다. <U>위와 기능상 차이점은 존재하지 않는다 봐도 됩니다.</U>

```java:main.dart
void main() {
  String name = 'John Doe';
  int age = 30;
  double height = 1.75;
  bool isStudent = false;

  print('Name: $name');
  print('Age: $age');
  print('Height: $height');
  print('Is Student: $isStudent');
}
```

# final - 런타임 불변 값값

**Dart** 에서 상수를 선언하기 위해 두가지 Keyword 를 사용할 수 있습니다. `final` and `const`

그중 `final` 은 <R>한 번만 값을 할당할 수 있는 변수</R>를 선언할 때 사용합니다. 한 번 값을 할당하면 **이후에는 변경할 수 없**습니다.

중요한 점은, 값을 **언제** 할당하느냐인데, `final` 은 **런타임에 값이 정해지는 상수**입니다.

```java:final.dart
final currentTime = DateTime.now(); // 현재 시간으로 초기화
```

> 메모리 할당
> <br/>
> `final` 변수는 **힙(Heap)** 메모리 저장되며, 더 이상 참조되지 않으면 **GC(Garbage Collector)**에 의해 자동 해제됩니다.


| 속성 | 설명 |
|---|---|
| 초기화 시점 | 런타임 |
| 메모리 위치 | 힙 (Heap) |
| 가비지 컬렉션 | 사용되지 않으면 GC가 해제 |
| 사용 예시 | 사용자 입력, API 응답 등 런타임 값 |

# const - 컴파일 타임 상수

`const` 는 <R>컴파일 타임에 값이 완전히 결정되는 상수</R>를 만들 때 사용합니다. 코드가 실행되기 이전부터 값이 정해져 있어야 하며, 불변이며 절대 바뀌지 않습니다.

```java:const.dart
const pi = 3.14159;
const api_url = 'https://api.example.com';
```

> 추가 정보
> <br/>
> 동일한 `const` 값은 하나의 인스턴스만 메모리에 존재합니다. 여러 곳에서 같은 값을 사용하더라도, **Dart** 는 하나만 메모리에 유지하며 이를 **canonicalization(정규화)**라고 부릅니다.
> <br/>
> `const` 는 프로그램 실행 전에 고정된 메모리에 저장되며, **GC의 대상이 아니고**, 프로그램 종료 시까지 유지됩니다.


| 속성 | 설명 |
|---|---|
| 초기화 시점 | 컴파일 타임 |
| 메모리 위치 | 고정된 메모리 영역 (힙 외부) |
| 가비지 컬렉션 | GC 대상 아님 (프로그램 종료 시까지 유지) |
| 사용 예시 | 상수 값, 불변 데이터 등 |
