# var Keyword

**Dart** 는 기본적으로 `var` 키워드를 사용하여 변수를 선언합니다. **Dart** 는 정적 타입 언어지만 `var` 키워드를 사용하면 타입을 명시하지 않고 변수를 선언할 수 있습니다.

하지만 그렇다고 **Type** 이 없다는 뜻이 아님니다. <R>Dart 는 타입 추론을 통해 변수의 타입을 자동으로 결정합니다.</R> 예를 들어, 아래와 같이 변수를 선언할 수 있습니다.

```java:main.dart
void main() {
  var name = 'John Doe'; // String
  var age = 30; // int
  var height = 1.75; // double
  var isStudent = false; // bool

  print('Name: $name');
  print('Age: $age');
  print('Height: $height');
  print('Is Student: $isStudent');
}
```

vscode 에서 name 변수에 마우스를 올리면 `String` 이라는 타입을 확인할 수 있습니다.

void main() {

# 명시적 타입 선언

**C++** 나 **Java** 처럼 명시적으로 타입을 선언할 수 있습니다. <U>위와 기능상 차이점은 존재하지 않는다 봐도 됩니다.</U>

```java:main.dart
void main() {
  String name = 'John Doe';
  int age = 30;
  double height = 1.75;
  bool isStudent = false;

  print('Name: $name');
  print('Age: $age');
  print('Height: $height');
  print('Is Student: $isStudent');
}
```

# final Keyword

Dart 에서 상수를 선언하는 방법은 2가지가 존재합니다. `final` 과 `const` 키워드 입니다.

먼저 `final` 키워드는 <R>런타임에 값을 할당할 수 있는 상수를 선언합니다.</R> 한번 값이 할당되면 변경할 수 없습니다. 예를 들어, 아래와 같이 `final` 키워드를 사용하여 상수를 선언할 수 있습니다.

```java:final.dart
void main() {
  final int year;
  year = 2023; // 런타임에 값을 할당

  print('Year: $year');
}
```

# const Keyword


`const` 키워드는 <R>컴파일 타임에 값을 할당 해야 하는 상수를 선언합니다.</R> 컴파일 타임에 값을 할당 한다는 의미는 <U>선언과 동시에 값이 정해져야 한다는 뜻입니다.</U>

예를 들어, 아래와 같이 `const` 키워드를 사용하여 상수를 선언할 수 있습니다.

```java:const.dart
void main() {
  const int year = 2023; // 컴파일 타임에 값을 할당

  print('Year: $year');
}
```

만약 `const` 키워드로 선언된 상수에 런타임에 값을 할당하려고 하면 에러가 발생합니다. 아래는 그 예시입니다.

```java:const.dart
void main() {
  const int year;
  year = 2023;

  print('Year: $year');
}
```
