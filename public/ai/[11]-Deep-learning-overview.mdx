# Q-table의 한계
Q-table은 상태의 수가 많아지면 너무 많은 메모리를 차지하게 됩니다. 이를 해결하기 위해 Q-table을 함수로 근사하는 방법이 있습니다. 이를 Function Approximation이라고 합니다. 이는 Q-table을 함수로 근사하는 것이기 때문에 Q-table의 한계를 극복할 수 있습니다.

# Deep learning을 위한 수학
이 책에선 딥하게 들어가지 않고, $y = W * x + b$ 이정도의 수학 정도만 이해해도 충분히 딥러닝을 이해할 수 있도록 구성했습니다. 정확히는 수학적 지식보단 프로그래밍 적인 지식이 더 중점이 돼는 책입니다.

하지만, 딥러닝을 이해하기 위해서는 미적분, 선형대수, 확률과 통계, 최적화 등의 수학적 지식이 필요합니다. 이 책을 읽고나서 더 깊게 공부하고 싶다면, 이 수학적 지식을 더 깊게 공부하시는 것을 추천드립니다.

Deep Learning을 구현하기 위해 PyTorch를 사용합니다. PyTorch는 딥러닝 프레임워크로, 텐서 연산을 위한 다양한 기능을 제공합니다. PyTorch를 사용하면 딥러닝 모델을 쉽게 구현할 수 있습니다. 앞으로 천천히 PyTorch를 사용해보면서 딥러닝을 구현해보겠습니다.

# Linear Regression
기본적인 딥러닝에 대해 소개했습니다. 이젠 기본적인 선형 회귀에 대해 알아보겠습니다. 선형 회귀는 가장 기본적인 머신러닝 알고리즘 중 하나입니다. 선형 회귀는 데이터의 경향성을 가장 잘 설명하는 하나의 직선을 찾는 알고리즘입니다.

먼저 이러한 데이터가 있다 가정해 보겠습니다.
| x(hours) | y(score) |
|----------|----------|
| 10       | 90       |
| 9        | 80       |
| 3        | 50       |
| 2        | 30       |

이 데이터는 x 시간 공부를 했을 때 y 점수를 얻었다는 데이터입니다. 이 데이터를 이용하여 Super vised learning을 한다 가정해 보겠습니다. 결과적으로 0 ~ 100 까지의 점수를 예측하는것이 목표이고 이러한 형태의 ML을 Super vised learning 중에서도 Regression이라고 합니다.

이러한 트레이닝 데이터를 가지고 train을 수행하면 regression 모델이 만들어 집니다. 쉽게 말해 그냥 Agent를 만든다고 생각하시면 됩니다. 그리고 이후 7시간이라는 IN을 던지면 regression은 학습된 데이터를 기반으로 적절한 OUT을 내놓습니다.

이러한 방식을 Linear Regression 이라고 합니다.

설명을 간단히 하기 위해 테이블을 더 간단히 하겠습니다. 이 모델을 가지고 regression 모델을 만들려고 하는 겁니다.

| x | y |
|---|---|
| 1 | 1 |
| 2 | 2 |
| 3 | 3 |
