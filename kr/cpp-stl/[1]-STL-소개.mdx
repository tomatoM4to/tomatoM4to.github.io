# STL (Standard Template Library)
**표준 템플릿 라이브러리(Standard Template Library, STL)**는 **generic class 와 generic function 들의 모음**이라 할수 있습니다.

generic 이란 무엇일까요? generic은 **일반적인** 이라는 뜻을 가지고 있습니다. 이어서 **일반적인** 이란 의미는 사전적으로 **모든것에 적용할수 있는** 이라는 뜻을 가지고 있습니다.

프로그래밍 언어적으로 이해해보면, generic class와 generic function은 <R>여러 타입을 동시에 커버 가능한 class와 function</R> 이라고 이해하면 됩니다.

그리고 `template`은 generic class와 generic function을 만들기 위한 **keyword**입니다. 마치 `class`나 `function`과 같은 키워드처럼요.
