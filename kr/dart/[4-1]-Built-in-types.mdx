# Built-in types

**기본 내장 타입** 혹은 **Built-in types** 는 Dart 에서 제공하는 기본 데이터 타입니다. 쉽게말해 그냥 `int`, `double`, `String` 등과 같은 타입을 말합니다. 먼저 직관적으로 바로 이해할수 있는 타입들부터 살펴보겠습니다.

1. `int` : 정수형 타입
2. `double` : 실수형 타입
3. `String` : 문자열 타입
4. `bool` : 불리언 타입 (true/false)
5. `List` : 리스트 타입 (배열과 비슷)
6. `Map` : 맵 타입 (key-value 쌍으로 이루어진 <U>데이터 구조</U>)
7. `Set` : 집합 타입 (중복을 허용하지 않는 <U>데이터 구조</U>)
8. `Null` : null 타입 (값이 없음을 나타내는 타입)

이 밖에도 `Runes`, `Symbol`, `Records`, `Function` 같은 타입도 존재하지만, 자주 사용하지도 않고, 한번에 설명하기엔 햇갈리기만 하기 떄문에 나중에 자세히 다루겠습니다.

중요한 점은 <R>이러한 방식의 타입 선언엔 객체를 생성하는 능력을 포합합니다!!!</R>

Dart 공식문서에서 이러한 설명이 존재합니다.

> Uninitialized variables that have a nullable type have an initial value of null. Even variables with numeric types are initially null, because numbers—like everything else in Dart—are objects.
> <br/>
> [https://dart.dev/language/variables](https://dart.dev/language/variables)

요약하면 다음과 같습니다. <R>Dart 는 숫자 조차도 Object로 취급한다.</R> 즉, `int` 나 `double` 타입도 **Object**로 취급되며, 이러한 객체들은 메모리에서 공간을 차지합니다. 따라서 <U>Dart 에서 제공하는 기본 내장 타입은 모두 **Object**입니다.</U>

결국 `int i = 0;` 이란 간단한 코드는 내부적이로로 이렇게 동작하게 됩니다.
1. 정수 `1` 은 `int` 타입의 객체로 힙(Heap) 메모리에 저장
2. 변수 `i` 는 `int` 타입의 객체(Object) 를 참조하는 포인터(Reference) 역할을 함


# Dart 에서 숫자가 객체임을 확인해보자

`int` 가 **Object** 라면 **Method**를 호출할 수 있어야 겠죠? 실제로 `int` 객체도 **Method**를 사용할 수 있습니다. 아래 코드에서 `num.isEven` 이나 `num.toDouble()` 처럼 `int` 객체에 **Method**를 호출하는걸 볼 수 있습니다.

```java:main.dart
void main() {
  int num = 42;
  print(num.isEven);  // true (짝수인지 확인하는 Method)
  print(num.toDouble()); // 42.0 (double로 변환)
}
```

혹은 이런식으로도 확인이 가능합니다.

```java:main.dart
void add(int i) {
  i++;
}

void main() {
  int out = 2025;

  print(out);  // 2025
  add(out);
  print(out);  // 2025 (out은 여전히 2025)
}
```

함수가 호출되었지만, `out` 의 값은 변하지 않았습니다. `int` 타입은 **Pass by Reference** 방식으로 동작한다는걸 확인할 수 있습니다.

# 이밖의 타입들

이밖에도 Dart 에는 <R>특별한 역할을 수행하는 타입들</R>이 존재합니다.

이들은 자주 사용되지는 않지만, 알아두면 유용한 타입들이긴 한데, 그렇게 많이 쓰지 않기도 하고, 완전히 이해하기 위해선 **function**, **class**, **generic** 등과 같은 개념과 함께 이해해야 하므로, 나중에 해당 주제를 다룰때 다시 설명하겠습니다.

일단 쉽게 이해되는 타입들부터 설명하겠습니다.

1. `Object` : `Null` 을 제외한 <U>모든 타입의 부모 클래스</U>입니다. Dart의 모든 값(객체)은 `Object` 타입으로 취급됩니다.
2. `dynamic`: <U>정적 검사를 사용하지 않는 타입</U> 입니다. 일반적으로 `Object` 또는 `Object?` 대신 사용합니다.
3. `void`: 함수 옆에만 붙어서 타입처럼 안생겼지만, 이 역시 Object 타입의 하위 타입입니다. 값이 사용되지 않음을 나타내고 반환 유형으로 사용됩니다.

일단 이정도만 알아둬도 Dart 에서 제공하는 기본 타입의 대부분을 이해한다 볼수 있습니다. 사실 절반정도긴 한데, 나머진 사용 빈도수가 매우 적어요.
