# 산술 연산자

몫을 반환하는 `~/` 연산자가 있다는걸 제외하면 다른 언어와 동일시 합니다.

참고로 연산자 우선순위는 `*`, `/`, `~/`, `%` **>** `+`, `-` 입니다. 그리고 연산자 우선순위가 같을 경우 **왼쪽에서 오른쪽으로 진행**합니다.

| 기호 | 의미 |
|---|---|
| `*` | 곱하기 |
| `/` | 나누기 |
| `%` | 나머지 |
| `~/` | 정수를 반환하는 나누기 연산자(몫) |
| `+` | 더하기 |
| `-` | 빼기 |

```dart:main.dart
void main() {
  int a = 10;
  int b = 3;
  print(a + b);
  print(a - b);
  print(a * b);
  print(a / b);
  print(a ~/ b);
  print(a % b);

  // 연산자 우선순위가 같으니 왼쪽에서 오른쪽으로 진행
  print(a ~/ b * 2);
}
```


# 증감 연산자

접미사, 접두사 증감 연산자 모두 지원 합니다. **접미사 증감 연산자**는 연산 **후**에 값을 **변화**시키고, **접두사 증감 연산자**는 연산 **전**에 값을 **변화**시킵니다.

Dart 의 모든 연산자 중 **가장 우선순위가 높은** 특징을 가지고 있습니다.

| 기호 | 의미 |
|---|---|
| `i++` | `var = var + 1` |
| `i--` | `var = var - 1` |
| `++i` | `var = var + 1` |
| `--i` | `var = var - 1` |

# 비교 연산자가

Dart 는 모든것이 객체지만, 일반적인 프로그래밍 언어처럼 비교연산자를 사용할 수 있습니다. 내부적으로 다 정의가 되어 있거든요. 자세한건 class 를 다룰때 설명하겠습니다.

| 기호 | 의미 |
|---|---|
| `==` | 같음 |
| `!=` | 다름 |
| `>` | 크다 |
| `<` | 작다 |
| `>=` | 크거나 같다 |
| `<=` | 작거나 같다 |

정말 간혹 객체가 같은 주소를 공유하는 같은 인스턴스 인지 확인해야 할때가 있을수 있습니다. 이럴땐 `identical()` 함수를 사용하면 됩니다. 이 함수는 두 객체가 같은 주소를 공유하는지 확인합니다. 또 기본적으로

```dart:main.dart
void main() {
  List<int> a = [1, 2, 3];
  List<int> b = [1, 2, 3];

  print(a == b); // false
  print(identical(a, b)); // false
}
```
