# 특별한 역할을 하는 타입들

이밖에도 Dart 에는 <R>특별한 역할을 수행하는 타입들</R>이 존재합니다.

이들은 자주 사용되지는 않지만, 알아두면 유용한 타입들이긴 한데, 그렇게 많이 쓰지 않기도 하고, 완전히 이해하기 위해선 **function**, **class**, **generic** 등과 같은 개념과 함께 이해해야 하므로, 나중에 해당 주제를 다룰때 다시 설명하겠습니다.

일단 쉽게 이해되는 타입들부터 설명하겠습니다.

1. `Object` : `Null` 을 제외한 <U>모든 타입의 부모 클래스</U>입니다. Dart의 모든 값(객체)은 `Object` 타입으로 취급됩니다.
2. `dynamic`: <U>정적 검사를 사용하지 않는 타입</U> 입니다. 일반적으로 `Object` 또는 `Object?` 대신 사용합니다.
3. `void`: 함수 옆에만 붙어서 타입처럼 안생겼지만, 이 역시 Object 타입의 하위 타입입니다. 값이 사용되지 않음을 나타내고 반환 유형으로 사용됩니다.

일단 이정도만 알아둬도 Dart 에서 제공하는 기본 타입의 대부분을 이해한다 볼수 있습니다. 사실 절반정도긴 한데, 나머진 사용 빈도수가 매우 적어요.
