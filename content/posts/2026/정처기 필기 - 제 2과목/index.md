## 트리

![tree](https://media.geeksforgeeks.org/wp-content/uploads/20251006175023711337/1.webp)

* 노드(node) : 트리의 원소, 15, 35, 3, 1, 10, 6, 9, ..., 7
* 간선(edge) : 노드를 연결하는 선
* 루트노드(root node) : 15
* 조상노드(ancestor node) : 루트에서 주어진 노드까지의 경로상에 있는 모든 노드
  * 1의 조상은 3, 35, 15
* 자손노드(descendant node) : 같은 부모를 공유하는 노드
  * 1, 10, 6 은 35의 자손
* 형제노드(sibling node) : 같은 부모 노드의 자식 노드들
  * 1과 10, 5와 7
* 단말노드(leaf node) : 자식 노드가 없는 노드
  * 1, 10, 6, 9, 8, 4, 41, 7
* 서브트리(subtree) : 부모 노드와 연결된 간선을 끊었을 때 생성되는 트리, 노드와 그 모든 자손 노드는 서브트리를 형선한다.
* 차수(degree)
  * 노드의 차수 : 노드에 연결된 자식 노드의 수, 15의 차수는 3
  * 트리의 차수 : 트리에 있는 노드의 차수중 가장 큰 값, 3
* 자식노드(child node) : 15의 자식 노드는 35, 9, 40
높이(depth, level)
  * 노드의 높이 : 루트에서 노드에 이르는 간선의 수, 15는 0, 9는 1, 1은 3
  * 트리의 높이 : 트리의 있는 노드의 높이 중 가장 큰 값, 3

## 정렬

### 버블 정렬 (Bubble Sort)
총 `n-1` 회전이 필요하며 `O(n^2)` 의 시간 복잡도를 가짐

* 1회전 `5, 1, 7, 10, 2` -> `1, 5, 7, 2, 10` 10 고정
* 2회전 `1, 5, 7, 2, 10` -> `1, 5, 2, 7, 10` 7, 10 고정
* 3회전 `1, 5, 2, 7, 10` -> `1, 2, 5, 7, 10` 5, 7, 10 고정
* 4회전 `1, 2, 5, 7, 10` -> `1, 2, 5, 7, 10` 2, 5, 7, 10 고정

example code
```python
def bubble_sort(arr):
    n = len(arr)
    for i in range(n):
        for j in range(0, n-i-1):
            if arr[j] > arr[j+1]:
                arr[j], arr[j+1] = arr[j+1], arr[j]
    return arr
```

### 삽입 정렬 (Insertion Sort)
좌, 우로 구분하여 좌측은 정렬된 부분, 우측은 정렬되지 않은 부분으로 나누고, 우측에서 하나씩 요소를 선택하여 좌측의 적절한 위치에 삽입하는 방식으로 정렬하는 알고리즘

총 `n-1` 회전이 필요하며 `O(n^2)` 의 시간 복잡도를 가짐

* 1회전 `5, 1, 7, 10, 2` -> `1, 5, 7, 10, 2`, 5와 1을 비교, `1, 5` 를 정렬되었다 고 생각
* 2회전 `1, 5, 7, 10, 2` -> `1, 5, 7, 10, 2`, 5와 7을 비교, `1, 5, 7` 을 정렬되었다 고 생각
* 3회전 `1, 5, 7, 10, 2` -> `1, 5, 7, 10, 2`, 7과 10을 비교, `1, 5, 7, 10` 을 정렬되었다 고 생각
* 4회전 `1, 5, 7, 10, 2` -> `1, 2, 5, 7, 10`, 10과 2를 비교, 7과 2를 비교, 5와 2를 비교하여 2를 1과 5 사이에 삽입

### 선택 정렬 (Selection Sort)
`0 + i` 번째 요소부터 `n-1` 번째 요소까지의 요소 중에서 가장 작은 요소를 선택하여 `i` 번째 요소와 교환하는 방식으로 정렬하는 알고리즘

총 `n-1` 회전이 필요하며 `O(n^2)` 의 시간 복잡도를 가짐

* 1회전 `5, 1, 7, 10, 2` -> `1, 5, 7, 10, 2`, 1이 가장 작은 요소이므로 5와 교환
* 2회전 `1, 5, 7, 10, 2` -> `1, 2, 7, 10, 5`, 2가 가장 작은 요소이므로 5와 교환
* 3회전 `1, 2, 7, 10, 5` -> `1, 2, 5, 10, 7`, 5가 가장 작은 요소이므로 7과 교환
* 4회전 `1, 2, 5, 10, 7` -> `1, 2, 5, 7, 10`, 7이 가장 작은 요소이므로 10과 교환
