# Securing email
## 기밀성(confidentiality)
다른사람이 봐도 모르게 보내고 싶다, 대칭키 사용, 대칭키는 수신자의 public key로 암호

대칭키는 기껏해야 2~6bit, 공개키로 암호화하면 엄청 오래걸림? 숫자가 1000bit 까지 감 그래서 효율을 위해 대칭키로암호화

긴 암호는 대칭키로암호화하고 대칭키는 공갸키로 암호화

message integrity에는 해시 알고리즘이 필요
