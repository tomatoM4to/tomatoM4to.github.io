TCP/IP Stack

м§ҖлӮңлІҲ мЈјм ңм—җм„ң OSI 7кі„мёөм—җ лҢҖн•ҙм„ң м•Ңм•„ліҙм•ҳмҠөлӢҲлӢӨ. н•ҳм§Җл§Ң OSI 7кі„мёөмқҖ л„ӨнҠёмӣҢнҒ¬ мӢңмҠӨн…ң кө¬м„ұмқ„ мң„н•ң лІ”мҡ©м Ғмқҙкі  к°ңл…җм Ғмқё лӘЁлҚё мқҙкі , TCP/IPлҠ” мқён„°л„·мқҙ л°ңлӘ…лҗҳл©ҙм„ң н•Ёк»ҳ к°ңл°ңлҗң Protocol Stack мһ…лӢҲлӢӨ. к·ёлҹ¬лҜҖлЎң мӢӨм ң мқён„°л„·мқҖ OSI 7кі„мёөмқҙ м•„лӢҢ TCP/IP Stackмқ„ мӮ¬мҡ©н•ҳкі  мһҲмҠөлӢҲлӢӨ.

к°ңл…җкіј к·ём—җ л”°лҘҙлҠ” м§„м§ң мӮ¬мӢӨмқҖ мЎ°кёҲм”© лӢӨлҘҙлӢӨлҠ”кІғмқ„ м•Ңм•„л‘җмӢңл©ҙ мўӢмқ„кұ° к°ҷмҠөлӢҲлӢӨ. OSI7кі„мёөмқҖ мҷ„м „нһҲ к°ңл…җм Ғмқё лӘЁлҚёмқҙкі , TCP/IP StackмқҖ мӢӨм ңлЎң мӮ¬мҡ©лҗҳлҠ” Protocol Stackмһ…лӢҲлӢӨ.

мқҙ л‘җк°ңмқҳ лӘЁлҚёмқҖ кҙҖлҰ¬н•ҳлҠ” мЈјмІҙлҳҗн•ң лӢӨлҰ…лӢҲлӢӨ. OSI 7кі„мёөмқҖ ISO(көӯм ңн‘ңмӨҖнҷ”кё°кө¬), IECм—җм„ң кҙҖлҰ¬н•ҳкі  мһҲкі , TCP/IP StackмқҖ IETF(Internet Engineering Task Force)м—җм„ң кҙҖлҰ¬н•ҳкі  мһҲмҠөлӢҲлӢӨ.

мқҙ л‘җк°ңмқҳ лӘЁлҚёмқҖ м–ҙлҠҗм •лҸ„ нҳёнҷҳм„ұмқҙ мһҲмҠөлӢҲлӢӨ. к·ёлҹ¬лҜҖлЎң OSI 7кі„мёөмқ„ мқҙн•ҙн•ҳкі  мһҲлӢӨл©ҙ TCP/IP StackлҸ„ мүҪкІҢ мқҙн•ҙн•  мҲҳ мһҲмҠөлӢҲлӢӨ. н•ңлІҲ TCP/IP Stackм—җ лҢҖн•ҙм„ң м•Ңм•„ліҙкІ мҠөлӢҲлӢӨ.

Application LayerлҠ” л§җ к·ёлҢҖлЎң Applicationмқҙ кө¬нҳ„/кҙҖлҰ¬ н•ҳлҠ” Layerмһ…лӢҲлӢӨ. л”°лқјм„ң мқҙ Layerмқҳ мЈј лӘ©м ҒмқҖ м•„лһҳмӘҪм—җм„ң м ңкіөн•ҳлҠ” Network кё°лҠҘмқ„ мӮ¬мҡ©н•ҳлҠ”кІғмқҙ мЈј лӘ©м Ғмқҙ мһҲмҠөлӢҲлӢӨ.

TCP/IPлҘј лҚ” мһҗм„ён•ҳкё° лӢӨлЈЁкё° м „ Socketкіј Portм—җ лҢҖн•ҙм„ң м•Ңм•„ліҙкІ мҠөлӢҲлӢӨ. Socketкіј Portлҳҗн•ң TCP/IPк°Җ к°ңл°ңлҸјлҠ” кіјм •м—җм„ң лӮҳмҳЁ к°ңл…җл“Өмқҙкё° л•Ңл¬ём—җ Protocolмқҙ Socketкіј PortлҘј м–ҙл–»кІҢ м •мқҳн–ҲлҠ”м§Җ м•Ңм•„лҙ…мӢңлӢӨ.

Port

Socketмқ„ м•Ңм•„ліҙкё°м „ TCP/IP Stackмқҙ нҷңл°ңн•ҳкІҢ к°ңл°ңлҗң мӢңкё°мқё 1970 ~ 1980л…„лҢҖлЎң лҸҢм•„к°Җ ліҙкІ мҠөлӢҲлӢӨ. мқҙл•ҢлҠ” TCP/IPк°Җ мҷ„м „нһҲ м •лҰҪлҗҳм§Җ м•Ҡм•ҳкё° л•Ңл¬ём—җ, м—¬лҹ¬ к°ңл…җл“Өмқҙ лӮҳмҳӨлҚҳ мӢңкё°мҳҖлҠ”лҚ°мҡ” мқҙм ң н•ңлІҲ мқҙл•ҢмӮ¬лһҢл“Өмқҙ н–ҲлҚҳ кі лҜјмқ„ м•Ңм•„ліҙкІ мҠөлӢҲлӢӨ.

мІ«лІҲм§ёлЎң, SystemмғҒм—җм„ң лҸҷмһ‘н•ҳлҠ” м–ҙл– н•ң ProcessAк°Җ мқён„°л„· мғҒмқҳ лҳҗлӢӨлҘё ProcessBмҷҖ лҚ°мқҙн„°лҘј мЈјкі л°ӣкі  мӢ¶лӢӨл©ҙ ProcessAмҷҖ Systemмқҙ м–ҙл–»кІҢ нҶөмӢ н•  м–ҙл– н•ң лҚ°мқҙн„° нҶөлЎңк°Җ мһҲм–ҙм•ј н• кІғмһ…лӢҲлӢӨ. к·ёлһҳм•ј Systemмқҙ ProcessBк°Җ мҶҚн•ң Systemм—җ лҚ°мқҙн„°лҘј м „мҶЎн•  мҲҳ мһҲмқ„кІғмһ…лӢҲлӢӨ. мқҙл ҮкІҢ ProcessмҷҖ м—°кІ°ehls data path нҳ№мқҖ data channelмқ„ Portлқј л¶ҖлҰ…лӢҲлӢӨ.

лҳҗ н•ң SystemмқҖ м—¬лҹ¬к°ңмқҳ ProcessлҘј нҸ¬н•Ён•ҳлҠ” кІҪмҡ°к°Җ л§ҺмҠөлӢҲлӢӨ. к·ёлҹ¬лҜҖлЎң Port лҳҗн•ң м—¬лҹ¬к°ң мЎҙмһ¬ н•ҳкі  к°Ғ ProcessлҠ” PortлҘј нҶөн•ҙ лҚ°мқҙн„°лҘј мЈјкі  л°ӣмҠөлӢҲлӢӨ. мқҙл•Ң PortлҠ” 16bitлЎң н‘ңнҳ„лҗҳл©°, 0 ~ 65535к№Ңм§Җ мӮ¬мҡ©н•  мҲҳ мһҲмҠөлӢҲлӢӨ. мқҙл•Ң 0 ~ 1023к№Ңм§ҖлҠ” Well-Known Portлқјкі  л¶ҲлҰ¬л©°, 1024 ~ 49151к№Ңм§ҖлҠ” Registered Port, 49152 ~ 65535к№Ңм§ҖлҠ” Dynamic Portлқјкі  л¶ҲлҰҪлӢҲлӢӨ.

TCP Protocolмқҳ н•„мҡ”м„ұ

мқҙм ң Portм—җ лҢҖн•ҙм„ң м•Ңм•„ліҙм•ҳмңјлӢҲ, мқҙ м§ҖмӢқмқ„ л°”нғ•мңјлЎң 1970 ~ 1980л…„лҢҖлЎң лҸҢм•„к°Җ лӢ№мӢң л„ӨнҠёмӣҢнҒ¬лҘј м—°кө¬н•ҳлҚҳ кіјн•ҷмһҗл“Өмқҙ м–ҙл– н•ң кі лҜјмқ„ н–ҲлҠ”м§Җ м•Ңм•„ліҙкІ мҠөлӢҲлӢӨ. лЁјм Җ мң„мҷҖ л§Ҳм°¬к°Җм§ҖлЎң ProcessAмҷҖ ProcessBк°Җ лҚ°мқҙн„°лҘј мЈјкі  л°ӣлҠ” мғҒнҷ©мқ„ к°Җм •н•ҙліҙкІ мҠөлӢҲлӢӨ. нҳ„мһ¬ TCP/IP Stackм—җм„ң кі л ӨлҸјм–ҙ мһҲлҠ” л¶Җл¶„мқҖ лӢӨмқҢкіј к°ҷмҠөлӢҲлӢӨ.

л‘җ Processк°Җ мһҲкі  ProcessлҠ” Systemкіј PortлЎң м—°кІ°лҸјм–ҙ мһҲмҠөлӢҲлӢӨ. мқҙм ң Internet Layerм—җм„ң л‘җ нҳёмҠӨнҠё мӮ¬мқҙмқҳ мөңм Ғмқҳ кІҪлЎңлҘј кІ°м •н•ҳкі , Link Layerм—җм„ң л¬јлҰ¬м Ғмқё м—°кІ°мқ„ лӢҙлӢ№н•©лӢҲлӢӨ. мқҙл•Ң л‘җ Processк°Җ лҚ°мқҙн„°лҘј мЈјкі  л°ӣкё° мң„н•ҙм„ңлҠ” м–ҙл– н•ң н”„лЎңнҶ мҪңмқҙ н•„мҡ”н• к№Ңмҡ”? мқҙл•Ң TCPк°Җ н•„мҡ”н•©лӢҲлӢӨ. мқҙл•Ң Internet Layerм—җм„ңлҠ” Internet Protocol(IP)лҘј мӮ¬мҡ©н•ҳкі , Link Layerм—җм„ңлҠ” Ethernetмқ„ мӮ¬мҡ©н•©лӢҲлӢӨ.

н•ҳм§Җл§Ң Internet Protocolм—җлҠ” мӨ‘мҡ”н•ң нҠ№м§•мқҙ мһҲмҠөлӢҲлӢӨ. мқҙ н”„лЎңнҶ мҪңмқҖ unreableн•ң н”„лЎңнҶ мҪңмһ…лӢҲлӢӨ. мҰү, лҚ°мқҙн„°лҘј м „мҶЎн• л•Ң лҚ°мқҙн„°к°Җ мҶҗмӢӨлҗҳкұ°лӮҳ, ліҙлӮҙлҠ” мҲңм„ңлҢҖлЎң л°ӣлҠ”мӘҪм—җм„ң л°ӣм§Җ лӘ»н• мҲҳлҸ„ мһҲмҠөлӢҲлӢӨ. Internet ProtolмқҖ н•ҳлӮҳмқҳ Hostм—җм„ң лӢӨлҘё HostлЎң лҚ°мқҙн„°лҘј ліҙлӮҙкё° мң„н•ҙ мөңм„ мқ„ лӢӨн•ҳкІ м§Җл§Ң 100%м•Ҳм „мқ„ ліҙмһҘн• мҲҳ м—ҶмҠөлӢҲлӢӨ. мқҙлҹ¬н•ң нҠ№м§•мқҖ м§ҖкёҲлҸ„ мң м§Җлҗҳкі  мһҲмҠөлӢҲлӢӨ.

к·ёл Үкё°м—җ Processк°„мқҳ нҶөмӢ м—җм„ң лҚ°мқҙн„°лҘј realiableн•ҳкІҢ мЈјкі  л°ӣмқ„ мҲҳ мһҲлҠ” Protocolмқҙ н•„мҡ”н•©лӢҲлӢӨ. к·ё кІ°кіјлЎң TCP(transmission Control protocol)мқҙ к°ңл°ңлҗҳм—ҲмҠөлӢҲлӢӨ. TCPлҠ” лҚ°мқҙн„°лҘј м „мҶЎн• л•Ң лҚ°мқҙн„°к°Җ мҶҗмӢӨлҗҳкұ°лӮҳ, мҲңм„ңк°Җ л’Өл°”лҖҢлҠ” л¬ём ңлҘј н•ҙкІ°н•ҳкё° мң„н•ҙ л§Ңл“Өм–ҙм§„ н”„лЎңнҶ мҪңмһ…лӢҲлӢӨ. TCPлҠ” IPмң„м—җм„ң лҸҷмһ‘н•ҳлҠ” Protocolмқҙм§Җл§Ң л…јлҰ¬м Ғмқё л°©лІ•мқ„ нҶөн•ҙ мӢӨм ң Processк°„ нҶөмӢ  мқҙ м•Ҳм •м ҒмңјлЎң мқҙлЈЁм–ҙм§ҖлҸ„лЎқ н•©лӢҲлӢӨ.

TCP

мқҙм ң TCPм—җ лҢҖн•ҙ мўҖлҚ” мһҗм„ёнһҲ мӮҙнҺҙліҙкІ мҠөлӢҲлӢӨ. лЁјм Җ Connectionм—җ лҢҖн•ң к°ңл…җл¶Җн„° мӮҙнҺҙліҙкІ мҠөлӢҲлӢӨ

Connection

ConnectionмқҖ мӮ¬м „м Ғмқё мқҳлҜёлЎ  м—°кІ°мқҙлһҖ лң»мқ„ к°Җм§Җм§Җл§Ң TCPм—җм„ң ConnectionмқҙлһҖ мўҖлҚ” кө¬мІҙм Ғмқё мқҳлҜёлҘј к°Җм§Җкі  мһҲмҠөлӢҲлӢӨ. TCPм—җм„ңмқҳ ConnectionмқҖ Processк°„мқҳ м•Ҳм •м Ғмқҙкі  л…јлҰ¬м Ғмқё нҶөмӢ  нҶөлЎңлҘј мқҳлҜён•©лӢҲлӢӨ. м—¬кё°м„ң л…јлҰ¬м Ғмқё мқҙлһҖ мқҳлҜёлҠ” к·ёлғҘ лӢЁмҲңн•ҳкІҢ мҪ”л“ңлЎң л§Ңл“Өм–ҙм§„ нҶөлЎң лқјлҠ” мқҳлҜёлқј мғқк°Ғн•ҙлҸ„ лҗ кұ° к°ҷмҠөлӢҲлӢӨ.

Three-way Handshake

мқҙм ң Connectionмқҳ к°ңл…җм—җ лҢҖн•ҙ м•Ңм•„ліҙм•ҳмңјлӢҲ TCPк°Җ м–ҙл–»кІҢ нҶөмӢ мқ„ н•ҳлҠ”м§Җ м•Ңм•„ліҙкІ мҠөлӢҲлӢӨ. лЁјм Җ TCP нҶөмӢ мқ„ н•ҳкё° мң„н•ҙ л‘җ ProcessлҠ” Connectionмқ„ л§әм–ҙм•ј н•©лӢҲлӢӨ. мқҙл•Ң Connectionмқ„ л§әкё° мң„н•ҙ Three-way HandshakeлқјлҠ” кіјм •мқ„ кұ°м№©лӢҲлӢӨ. Connectionмқҙ л§әм–ҙм§ҖкІҢ лҸјл©ҙ л‘җ ProcessлҠ” мқҙ Connection мң„м—җм„ң лҚ°мқҙн„°лҘј мЈјлҸ„ л°ӣмқ„ мҲҳ мһҲмҠөлӢҲлӢӨ.

Tree-way HandshakeлҘј н•ҳлҠ” мқҙмң лҠ” л‘җ Processк°Җ м•Ҳм •м ҒмңјлЎң лҚ°мқҙн„°лҘј мЈјкі  л°ӣкё° мң„н•ң Set up кіјм •мқҙлқј мғқк°Ғн•ҳл©ҙ лҗ кұ° к°ҷмҠөлӢҲлӢӨ.

к·ёлҰ¬кі  нҶөмӢ мқҙ лҒқлӮҳл©ҙ Connectionмқ„ мў…лЈҢн•ҙм•ј н•©лӢҲлӢӨ. мқҙл•Ң Connectionмқ„ мў…лЈҢн•ҳкё° мң„н•ҙ Four-way HandshakeлқјлҠ” кіјм •мқ„ кұ°м№©лӢҲлӢӨ. мқҙ кіјм •мқ„ нҶөн•ҙ Connectionмқҙ м•Ҳм „н•ҳкІҢ мў…лЈҢлҗ  мҲҳ мһҲмҠөлӢҲлӢӨ.

Four-way HandshakeлҘј н•ҳлҠ” мқҙмң лҠ” л‘җ Processк°Җ SystemмңјлЎңл¶Җн„° н• лӢ№л°ӣмқҖ м—¬лҹ¬ мһҗмӣҗл“Өмқ„ л°ҳнҷҳн•ҳкё° мң„н•ң кіјм •мқҙлқј мғқк°Ғн•ҳл©ҙ лҗ кұ° к°ҷмҠөлӢҲлӢӨ.

к°ҖлҒ” м–ҙл– н•ң Protocolмқ„ мӮҙнҺҙліҙл©ҙ Connection-oriented ProtocolмқҙлқјлҠ” мҡ©м–ҙлҘј ліјмҲҳ мһҲмҠөлӢҲлӢӨ. мқҙл•Ң Connection-oriented ProtocolмқҙлһҖ Connectionмқ„ л§әкі  лҚ°мқҙн„°лҘј мЈјкі  л°ӣлҠ” Protocolмқ„ мқҳлҜён•©лӢҲлӢӨ. TCPлҠ” Connection-oriented Protocolмқҙкё° л•Ңл¬ём—җ Three-way HandshakeмҷҖ Four-way HandshakeлҘј нҶөн•ҙ Connectionмқ„ л§әкі  мў…лЈҢн•©лӢҲлӢӨ.
