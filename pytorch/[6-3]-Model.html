모델 정의

Pytorch에선 NN을 구성할때 반드시 nn.Module을 상속받아야 합니다. 그리고 __init__과 forward를 반드시 구현해야 합니다. 커스텀 NN또한 마찬가지 입니다.

  • __init__(): 모델에서 사용딜 모듈과 활성화 함수 등을 정의합니다.
  • forward(): 모델에서 실행되어야 하는 연산을 정의합니다.

Example

import torch.nn as nn

class Net(nn.Module):
    def __init__(self, inputs):
        super(Net, self).__init__()
        self.layer = nn.Linear(inputs, 1)
        self.activation = nn.ReLU()

    def forward(self, x):
        x = self.layer(x) # layer 통과
        x = self.activation(x) # activation 통과
        return x

model = Net(1)
print(list(model.children())) # [Linear(in_features=1, out_features=1, bias=True), ReLU()]
print(list(model.modules()))

마지막 두번째 print 문을 통해 Net 클래스에 정의된 레이어들이 출력됩니다.

nn.Sequential 을 이용한 신경망 정의

위 코드를 보면 forward() 에서 연산을 하기 위해 layer를 여러개 불러와 복잡한 연산을 수행하게 됩니다. 하지만 이렇게 되면 코드가 복잡해져서 디버깅이 어려워집니다. 이럴때 nn.Sequential을 통해 가독성 높게 코드를 작성할수 있습니다.

import torch.nn as nn

class Net(nn.Module):
    def __init__(self, inputs):
        super(Net, self).__init__()
        self.layer = nn.Sequential(
            nn.Linear(inputs, inputs),
            nn.ReLU()
            nn.Linear(inputs, 1)
            nn.ReLU()
        )

    def forward(self, x):
        return self.layer(x)
