Style Guide

final - 런타임에 한 번만 결정

Dart 에서는 상수를 선언할 때 final 과 const 두 가지 키워드를 사용할 수 있습니다. 이 중 final은 한 번만 값을 할당할 수 있는 변수를 선언할 때 사용합니다. 한 번 값이 할당되면 그 이후에는 재할당할 수 없습니다.

여기서 중요한 포인트는 값이 언제 결정되는가입니다. final 은 런타임에 값이 정해지는 상수입니다. 실행 중 얻은 값을 저장할 수 있죠. 다만, final이 막는 것은 변수 자체의 재할당이며, 내부 데이터의 변경까지 막지는 않습니다.

final.dart

final은 주로 API 응답, 파일 읽기 결과, 현재 시간처럼 실행 중에 얻는 값을 저장할 때 사용합니다.

const - 컴파일 타임 상수

const 는 컴파일 타임에 값이 완전히 결정되는 상수를 선언할 때 사용합니다. 코드가 실행되기 이전부터 값이 고정되어 있으며, 한 번 정의된 뒤에는 절대 변경될 수 없습니다.

또한 const의 불변성은 변수 자체뿐 아니라, 내부에 포함된 모든 데이터도 const이어야 합니다.

canonicalization (정규화화)

const 변수는 canonicalization이라는 최적화 방식을 사용합니다. 같은 값을 가지는 const 객체들은 메모리에서 하나만 생성되어 자동으로 공유됩니다.

const-최적화
