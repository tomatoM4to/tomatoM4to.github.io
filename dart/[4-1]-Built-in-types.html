Built-in types

кё°ліё лӮҙмһҘ нғҖмһ… нҳ№мқҖ Built-in types лҠ” Dart м—җм„ң м ңкіөн•ҳлҠ” кё°ліё лҚ°мқҙн„° нғҖмһ…лӢҲлӢӨ. мүҪкІҢл§җн•ҙ к·ёлғҘ int, double, String л“ұкіј к°ҷмқҖ нғҖмһ…мқ„ л§җн•©лӢҲлӢӨ. лЁјм Җ м§ҒкҙҖм ҒмңјлЎң л°”лЎң мқҙн•ҙн• мҲҳ мһҲлҠ” нғҖмһ…л“Өл¶Җн„° мӮҙнҺҙліҙкІ мҠөлӢҲлӢӨ.

  1. int : м •мҲҳнҳ• нғҖмһ…
  2. double : мӢӨмҲҳнҳ• нғҖмһ…
  3. String : л¬ёмһҗм—ҙ нғҖмһ…
  4. bool : л¶ҲлҰ¬м–ё нғҖмһ… (true/false)
  5. List : лҰ¬мҠӨнҠё нғҖмһ… (л°°м—ҙкіј л№„мҠ·)
  6. Map : л§ө нғҖмһ… (key-value мҢҚмңјлЎң мқҙлЈЁм–ҙм§„ лҚ°мқҙн„° кө¬мЎ°)
  7. Set : м§‘н•© нғҖмһ… (мӨ‘ліөмқ„ н—Ҳмҡ©н•ҳм§Җ м•ҠлҠ” лҚ°мқҙн„° кө¬мЎ°)
  8. Null : null нғҖмһ… (к°’мқҙ м—ҶмқҢмқ„ лӮҳнғҖлӮҙлҠ” нғҖмһ…)

мқҙ л°–м—җлҸ„ Runes, Symbol, Records, Function к°ҷмқҖ нғҖмһ…лҸ„ мЎҙмһ¬н•ҳм§Җл§Ң, мһҗмЈј мӮ¬мҡ©н•ҳм§ҖлҸ„ м•Ҡкі , н•ңлІҲм—җ м„ӨлӘ…н•ҳкё°м—” н–Үк°ҲлҰ¬кё°л§Ң н•ҳкё° л–„л¬ём—җ лӮҳмӨ‘м—җ мһҗм„ёнһҲ лӢӨлЈЁкІ мҠөлӢҲлӢӨ.

мӨ‘мҡ”н•ң м җмқҖ мқҙлҹ¬н•ң л°©мӢқмқҳ нғҖмһ… м„ м–ём—” к°қмІҙлҘј мғқм„ұн•ҳлҠ” лҠҘл Ҙмқ„ нҸ¬н•©н•©лӢҲлӢӨ!!!

Dart кіөмӢқл¬ём„ңм—җм„ң мқҙлҹ¬н•ң м„ӨлӘ…мқҙ мЎҙмһ¬н•©лӢҲлӢӨ.

Uninitialized variables that have a nullable type have an initial value of null. Even variables with numeric types are initially null, because numbersвҖ”like everything else in DartвҖ”are objects.


https://dart.dev/language/variables

мҡ”м•Ҫн•ҳл©ҙ лӢӨмқҢкіј к°ҷмҠөлӢҲлӢӨ. Dart лҠ” мҲ«мһҗ мЎ°м°ЁлҸ„ ObjectлЎң м·Ёкёүн•ңлӢӨ. мҰү, int лӮҳ double нғҖмһ…лҸ„ ObjectлЎң м·Ёкёүлҗҳл©°, мқҙлҹ¬н•ң к°қмІҙл“ӨмқҖ л©”лӘЁлҰ¬м—җм„ң кіөк°„мқ„ м°Ём§Җн•©лӢҲлӢӨ. л”°лқјм„ң Dart м—җм„ң м ңкіөн•ҳлҠ” кё°ліё лӮҙмһҘ нғҖмһ…мқҖ лӘЁл‘җ Objectмһ…лӢҲлӢӨ.

кІ°көӯ int i = 0; мқҙлһҖ к°„лӢЁн•ң мҪ”л“ңлҠ” лӮҙл¶Җм ҒмқҙлЎңлЎң мқҙл ҮкІҢ лҸҷмһ‘н•ҳкІҢ лҗ©лӢҲлӢӨ.

  1. м •мҲҳ 1 мқҖ int нғҖмһ…мқҳ к°қмІҙлЎң нһҷ(Heap) л©”лӘЁлҰ¬м—җ м ҖмһҘ
  2. ліҖмҲҳ i лҠ” int нғҖмһ…мқҳ к°қмІҙ(Object) лҘј м°ёмЎ°н•ҳлҠ” нҸ¬мқён„°(Reference) м—ӯн• мқ„ н•Ё

Dart м—җм„ң мҲ«мһҗк°Җ к°қмІҙмһ„мқ„ нҷ•мқён•ҙліҙмһҗ

int к°Җ Object лқјл©ҙ MethodлҘј нҳём¶ңн•  мҲҳ мһҲм–ҙм•ј кІ мЈ ? мӢӨм ңлЎң int к°қмІҙлҸ„ MethodлҘј мӮ¬мҡ©н•  мҲҳ мһҲмҠөлӢҲлӢӨ. м•„лһҳ мҪ”л“ңм—җм„ң num.isEven мқҙлӮҳ num.toDouble() мІҳлҹј int к°қмІҙм—җ MethodлҘј нҳём¶ңн•ҳлҠ”кұё ліј мҲҳ мһҲмҠөлӢҲлӢӨ.

main.dart
void main() {
  int num = 42;
  print(num.isEven);  // true (м§қмҲҳмқём§Җ нҷ•мқён•ҳлҠ” Method)
  print(num.toDouble()); // 42.0 (doubleлЎң ліҖнҷҳ)
}

нҠ№мқҙн•ң нғҖмһ…л“Ө

мқҙл°–м—җлҸ„ Dart м—җлҠ” нҠ№лі„н•ң м—ӯн• мқ„ мҲҳн–үн•ҳлҠ” нғҖмһ…л“Өмқҙ мЎҙмһ¬н•©лӢҲлӢӨ. мқҙл“ӨмқҖ мһҗмЈј мӮ¬мҡ©лҗҳм§ҖлҠ” м•Ҡм§Җл§Ң, м•Ңм•„л‘җл©ҙ мң мҡ©н•ң нғҖмһ…л“Өмһ…лӢҲлӢӨ.

мӮ¬мӢӨ Null лҸ„ мқҙмӘҪм—җ н•ҙлӢ№н•ҳкёҙ н•ҳлҠ”лҚ°, л§ҺмқҖ н”„лЎңк·ёлһҳл°Қ м–ём–ҙк°Җ кұ°мқҳ н‘ңмӨҖмІҳлҹј м§Җмӣҗн•ҳкё°м—җ көімқҙ нҠ№мқҙн•ҳлӢӨ н‘ңнҳ„н•ҳм§Җ м•ҠкІ мҠөлӢҲлӢӨ.

  1. Object : Null мқ„ м ңмҷён•ң лӘЁл“  нғҖмһ…мқҳ л¶ҖлӘЁ нҒҙлһҳмҠӨмһ…лӢҲлӢӨ. мҰү, лӘЁл“  к°қмІҙлҠ” Object нғҖмһ…мңјлЎң м·Ёкёүлҗ©лӢҲлӢӨ.
  2. Object?: Null мқ„ н—Ҳмҡ©н•ҳлҠ” мҶҚм„ұмқҙ м¶”к°Җлҗң Object нғҖмһ…мһ…лӢҲлӢӨ. к·ёлҹ¬лҜҖлЎң null-check к°Җ н•„мҡ”н•  мҲҳ лҸ„ мһҲмҠөлӢҲлӢӨ.
  3. dynamic: м •м Ғ кІҖмӮ¬лҘј мӮ¬мҡ©н•ҳм§Җ м•ҠлҠ” нғҖмһ… мһ…лӢҲлӢӨ. мқјл°ҳм ҒмңјлЎң Object лҳҗлҠ” Object? лҢҖмӢ  мӮ¬мҡ©н•©лӢҲлӢӨ.
